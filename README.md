PhD Pod Booking System

A streamlined, mobile-first web application designed to manage bookings for the "PhD Pod" quiet space at Monash University. This system provides an intuitive interface for students to reserve slots and a comprehensive dashboard for administrators to monitor usage.

🚀 Key Features

For Students

Easy Booking: Simple form to reserve a time slot using Name, Student ID, and Monash Email.

Real-time Schedule: View availability instantly to avoid double bookings.

Self-Management: Students set a 4-digit PIN upon booking, allowing them to Reschedule or Cancel their own slots securely without admin intervention.

Mobile Optimized: Responsive design that works perfectly on smartphones via QR code access.

For Administrators (MGRO)

Secure Dashboard: Hidden access via a password-protected login.

Advanced Analytics:

Weekly Activity: Visual heatmap of the busiest days.

Time Preference: Breakdown of usage by Morning, Afternoon, and Evening.

User Activity: Leaderboard of most frequent users.

Utilization Stats: Total bookings and hours utilized.

Data Management: Filter data by Week, Month, or Custom Date Range.

Export Data: One-click download of all booking data to CSV (Excel) for reporting.

Admin Controls: Ability to delete any booking and change the admin password directly from the settings.

🛠️ Technology Stack

This application is built as a Single-File Solution for maximum portability and ease of deployment (e.g., GitHub Pages).

Frontend: React.js (via CDN)

Styling: Tailwind CSS (via CDN) for modern, responsive UI.

Backend & Database: Google Firebase (Firestore) for real-time data storage.

Authentication: Firebase Anonymous Auth & Custom Security Rules.
